#confluence

##Overview

Puppet module to install and manage Atlassian Confluence.

##Usage

```
class { 'confluence': }
```

##Limitations

The module has been tested on the following operating systems. Testing and patches for other platforms are welcome.

* Debian Linux 6.0 (Squeeze)
